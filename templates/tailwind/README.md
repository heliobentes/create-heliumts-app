# Tailwind HeliumTS Template

This folder contains the HeliumTS template with Tailwind CSS pre-configured.

## Template Contents

Place your Tailwind HeliumTS project files here:

- `package.json` - Project dependencies (includes Tailwind CSS)
- `vite.config.ts` - Vite configuration with HeliumTS plugin
- `tsconfig.json` - TypeScript configuration
- `tailwind.config.js` - Tailwind CSS configuration
- `postcss.config.js` - PostCSS configuration
- `index.html` - HTML entry point
- `src/App.tsx` - App shell component
- `src/index.css` - Global styles with Tailwind directives
- `src/pages/` - Pages directory
- `src/server/` - Server-side code
- `src/components/` - React components
